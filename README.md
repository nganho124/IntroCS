# Intro to Computer Science (CS50)

This repository contains my solutions and coursework for the Introduction to Computer Science curriculum. It covers fundamental concepts in programming, data structures, and web development.

## 📂 Project Structure

* **01. C** – Programming basics: variables, functions, and control flow.
* **02. Arrays** – Compiling, debugging, and memory layout.
* **03. Algorithms** – Searching (Linear, Binary) and Sorting (Bubble, Selection, Merge).
* **04. Memory** – Pointers, addresses, and dynamic memory allocation.
* **05. Data Structures** – Linked lists, hash tables, and tries.
* **06. Python** – Transitioning from C to high-level scripting.
* **07. SQL** – Relational databases and data management.
* **08. HTML/CSS/JS** – Front-end web development basics.
* **09. Flask** – Building web applications with Python.

## 🚀 Technologies Used

* **Languages:** C, Python, SQL, JavaScript
* **Tools:** SQLite, Flask, VS Code/Codespaces
* **Concepts:** Algorithmic complexity (Big O), Memory Management, Web Security
